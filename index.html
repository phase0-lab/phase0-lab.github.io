The Problem

AI changed how we code. Not how we plan.

Developers have Cursor and Claude Code. Product managers still spend 40-60% of their time on context archaeology — digging through Slack threads, meeting recordings, and scattered documents.

Existing AI tools help you write documents faster. They don't help you think clearer. Phase0 Lab builds tools that bridge the gap between discovery and implementation.

Tools

phase0-spec

Living specification system. Generate and maintain PRDs, user stories, and acceptance criteria from context.

phase0-discovery

Context aggregation agent. Collects signals from Slack, GitHub, meetings, and builds a product knowledge graph.

phase0-radar

Open-source competitive intelligence. Track GitHub releases, ProductHunt, changelogs, and blogs.

phase0-bridge

Planning to code. Decompose specs into tasks and feed them into Claude Code, Cursor, or Copilot.
